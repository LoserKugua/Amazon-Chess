### 代码相关

代码除了`.qrc`文件这种资源文件之外，<b>100%</b>采用C++编写(`std=c++17`)。

开发相关库为C++Qt，版本$6.10.1$。

### 功能介绍

菜单拆分成了主菜单和游戏内副菜单，符合现代游戏的GUI设计（

实现了新开始、存档、读档、悔棋功能，游戏读档采用读取`.txt`文件方式，读取一般比较智能，不会读取非游戏存档文件，以及不会成功读取损坏的存档文件。

GUI设计<del>(自认为)</del>比较好的点:

1. 按钮更现代化，普通情况、鼠标悬停、按下的时候颜色会加深，按下还会有文字浮动，更有“按钮”感
2. 棋盘色调有变化，棋子之类的主色调符合PPT内容，加上了阴影部分

大作业附加功能：

1. 双人对战模式
2. BGM & 可以自由随窗口缩放的背景
3. 可以悔棋
4. 有提示功能
5. 一个设置，调音量，切歌，换背景
